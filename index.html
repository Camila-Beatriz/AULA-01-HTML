AULA 02

Conhecendo tags, elementos, atributos e valores
    Tags - São os códigos ou conjuntos para marcar o início e o fim de um elemento
    Elementos - São os conteúdos dentro das tags, indicam ao que se refere a informação das tags, no caso o conteúdo de um parágrafo;
    Atributos - São as determinações do elemento que está na tag, por exemplo style para determinsr o estílo da formatação;
    Valor - É a definição do que deve ser realizado no atributo;
